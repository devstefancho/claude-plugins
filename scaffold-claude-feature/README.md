# Claude Code Generator Plugin

Generate Claude Code features with proper structure and best practices.

## Features

- **Slash Commands** - Reusable prompts
- **Skills** - Model-invoked capabilities
- **Subagents** - Specialized AI agents
- **Hooks** - Event handlers
- **Output Styles** - Custom system prompts

## Installation

```bash
/plugin install claude-code-generator-plugin@devstefancho-claude-plugins
```

## Usage

The skill is automatically invoked when you ask to create Claude Code features:

- "Create a new slash command for..."
- "Generate a skill that..."
- "Add a hook for..."

## Included Resources

- Reference documentation for each feature type
- Templates for quick generation
- Best practice guidelines
